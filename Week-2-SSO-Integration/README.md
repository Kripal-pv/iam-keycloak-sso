# 🌐 Week 2: SSO Integration using Keycloak

## 🎯 Objective
Integrate a web application with Keycloak using OpenID Connect (OIDC) to enable Single Sign-On.

---

## 🛠️ Tasks Completed
- Created OIDC client (my-app)
- Configured redirect URIs
- Developed Flask application
- Implemented login redirection to Keycloak
- Received authorization code after login

---

## 🔄 Authentication Flow
User → App → Keycloak → Login → Redirect → App

---

## 🧠 Concepts
- OpenID Connect (OIDC)
- Authorization Code Flow
- Single Sign-On (SSO)

---

## 🎯 Outcome
A working SSO system where authentication is handled by Keycloak.
